# Acceso a la base de datos
Desde la terminal en el Docker-desktop:
 `psql -U receta`       --> Conecta a la base de datos
 `\dt`                  --> Lista las tablas creadas
 `\d categorias`        --> Muestra la estructura de la tabla pedida
